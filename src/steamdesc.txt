Proof of concept chat box. Offline only at the moment, but whenever multiplayer/networking is a thing, I'll be sure to add it as soon as possible :).

[b]Features:[/b]
[list]
    [*]Colored usernames
    [*]Scrollable chat
    [*]Chat commands - try them out with .help
    [*]Text wrapping when typing and inside the message window
[/list]

[b]Controls:[/b]
[list]
    [*]T opens chat
    [*]CTRL is backspace (sorry, the backspace key isn't possible ATM)
    [*]RMB or ESC closes chat
[/list]

[/list]
